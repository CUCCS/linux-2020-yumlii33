## 安装asciinema

```
//安装
sudo apt-get install asciinema
//开始一个记录文件 test.cast
asciinema rec test.cast
//结束录制 ctrl+D or exit
//本地配置
asciinema auth
//上传
asciinema upload test.cast
```
